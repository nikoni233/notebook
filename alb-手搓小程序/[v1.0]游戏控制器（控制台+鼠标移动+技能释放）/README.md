## 项目目录结构
```
game_controller/
├── main.py                # 主程序，启动控制面板
├── controller.py          # 控制应用与操作的逻辑
├── gui.py                 # 控制面板界面
├── requirements.txt       # 项目所需的依赖库
└── README.md              # 项目说明
```

## 安装依赖项
```bash
pip install -r requirements.txt
```

## 启动
运行`main.py`
